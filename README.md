# 川大图书馆快速搜索

一个简洁的四川大学图书馆搜索工具，支持手机和电脑使用。

## 🚀 使用方法

### 在线使用
直接访问：https://[你的用户名].github.io/scu-library-search

### 本地使用
1. 下载 `scu-library-search.html`
2. 双击文件即可在浏览器中打开
3. 支持离线使用

### 手机使用
- **添加到主屏幕**：在浏览器中打开后，添加到主屏幕即可像App一样使用
- **分享功能**：可以分享给同学使用

## ✨ 功能特点
- 📱 响应式设计，完美适配手机和电脑
- 💾 本地存储搜索历史
- 🔍 一键跳转到川大图书馆搜索结果
- 🚀 无需安装，即开即用
- 📡 支持离线查看历史记录

## 🛠️ 技术栈
